📋 第一章：初入江湖

1注册和创建角色 
register 张无忌 123456    # 注册一个账号
login 张无忌 123456       # 登录已有账号

或者直接用演示账号：login demo some-secret（预设属性，10000 潜能）

2分配属性

创建角色后，输入角色名（必须是中文），然后分配 10 点自由属性：

set str 20    # 臂力（影响攻击力）
set con 20    # 根骨（影响 HP 上限）
set dex 15    # 身法（影响闪避、战斗速度）
set int 10    # 悟性（影响 MP 上限、练习效率）
done          # 确认完成创建
推荐新手上路加点： str=15, con=15（高攻击+高血量，打怪不死）
3熟悉基本命令 
look          # 查看四周（会显示地上物品和 NPC）
hp / score    # 查看状态（HP/MP/经验/潜能/等级/属性点）
level         # 查看等级、经验进度和属性点
help          # 显示所有命令
skills        # 查看已学武功
i             # 查看背包物品
who           # 查看在线玩家

📈 第二章：学习武功

武功分两类：基本武功随处可学，独门武功需先拜师。

learn 基本拳脚    # 重复输入升级，每次升 1 级
升级技巧：用 ↑↓ 箭头键调出历史命令，快速重复 learn！

🔹 练习武功（practice）

不想消耗潜能？可以自行练习已学会的武功：

practice 基本拳脚     # 不消耗潜能，提升 1 级
practice dodge        # 练习轻功
注意：练习有冷却时间，且武功等级不能超过 玩家等级 × 10。想要突破上限，先去升级！

🔹 打坐恢复（dazuo）

学会基本内功后，可以通过打坐把气血转化为内力：

learn 基本内功        # 先学会内功
dazuo 30              # 打坐 30 秒恢复内力
打坐时无法移动或战斗，血量会缓慢下降，内力会持续回复。

🎖 第三章：等级与属性点

江湖之路，等级是实力的重要标志。获得经验即可升级，每次升级奖励 2 点属性点。

level             # 查看当前等级、经验进度、属性点
tianfu str 2      # 分配 2 点属性点到臂力
tianfu 根骨 1     # 也可以写中文属性名
经验来源说明
杀怪山贼、野狼、黑熊等都会给经验和银两
任务说书人的送信任务给大量经验
绝招杀敌施展 perform 击杀也能获得经验
升级奖励：每升 1 级，气血上限和内力上限会按根骨/悟性提升，并获得 2 点自由属性点。

⚔ 第四章：打怪升级

山林中有主动攻击的野怪，杀死获得经验 + 潜能 + 银两。

野怪ID位置路径难度
山贼bandit山林·入口nnn易
野狼wolf山林·深处nnnn中
黑熊bear山林·洞穴nnnnw难
# 打山贼（新手首选）
n n n              # 走到山林入口
kill 山贼           # 发起战斗（也可以用英文 ID kill bandit）
# 自动回合制战斗，等几轮后山贼倒下
# 获得经验、潜能和银两！

# 逃跑（打不过就溜）
flee / tao
⚠ 注意：战死后会扣 10% 经验！血少的时候先用 exert heal 疗伤（消耗 30 内力，恢复 20% HP）。如果内力不足，远离战斗等自动恢复（每 5 秒恢复 3% HP / 4% MP）。

🦠 第五章：状态与解毒

某些怪物攻击会附带负面状态（如野狼、山贼的毒）。状态会在 score 中显示。

状态来源处理
中毒（poison）野狼、山贼等攻击概率附带use 解毒丸 或 exert dispel poison
score                    # 查看是否中毒
exert dispel poison      # 用内功逼毒（需要基本内功）
use 解毒丸                # 服用解毒丸直接清除
提示：解毒丸在主街药铺老板处购买（50 两银子）。

⚡ 第九章：进阶技巧

绝招（perform）

# 武功升到 Lv.10 后可施展绝招
learn 基本拳脚    × 10 次（升到 Lv.10）
perform 基本拳脚.黑虎掏心   # 施展绝招（消耗 20 内力，伤害 2 倍）

内功运用（exert / yun）

exert heal              # 疗伤：消耗 30 内力，恢复 20% HP
yun heal                # yun 是 exert 的别名
exert powerup           # 战力爆发：消耗 50 内力，30 秒内攻防提升
exert dispel poison     # 运功逼毒（需要基本内功）

独门绝招

各门派独门武功有更强的绝招（在师父房间学得）。学到 Lv.10 后 perform 武功名.招式名 施放。
