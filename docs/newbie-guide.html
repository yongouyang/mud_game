2分配属性

创建角色后，输入角色名（必须是中文），然后分配 10 点自由属性：

set str 20    # 臂力（影响攻击力）
set con 20    # 根骨（影响 HP 上限）
set dex 15    # 身法（影响闪避、战斗速度）
set int 10    # 悟性（影响 MP 上限、练习效率）
set per 12    # 容貌（影响部分 NPC 对话和武功学习）
set kar 10    # 福缘（影响随机事件运气）
done          # 确认完成创建
推荐新手上路加点： str=15, con=15（高攻击+高血量，打怪不死）。剩余点数可按喜好分给容貌或福缘。

🔹 练习武功（practice）

不想消耗潜能？可以自行练习已学会的武功：

practice 基本拳脚     # 不消耗潜能，提升 1 级
practice dodge        # 练习轻功
注意：练习有冷却时间，且武功等级不能超过 玩家等级 × 10。想要突破上限，先去升级！

🔹 打坐恢复（dazuo）

学会基本内功后，可以通过打坐把气血转化为内力：

learn 基本内功        # 先学会内功
dazuo 30              # 打坐 30 秒恢复内力
打坐时无法移动或战斗，血量会缓慢下降，内力会持续回复。

🎖 第三章：等级与属性点

江湖之路，等级是实力的重要标志。获得经验即可升级，每次升级奖励 2 点属性点。

level             # 查看当前等级、经验进度、属性点
tianfu str 2      # 分配 2 点属性点到臂力
tianfu 根骨 1     # 也可以写中文属性名
经验来源说明
杀怪山贼、野狼、黑熊等都会给经验和银两
任务说书人的送信任务给大量经验
绝招杀敌施展 perform 击杀也能获得经验
升级奖励：每升 1 级，气血上限和内力上限会按根骨/悟性提升，并获得 2 点自由属性点。

🦠 第五章：状态与解毒

江湖中处处凶险，怪物、毒物、恶劣天气都会让你染上负面状态。状态会在 score 中显示。

🧧 拍卖行（auction 命令）

⚔ 第十一章：进阶战斗机制

除了基础攻击，战斗还受武器、连招等机制影响：

机制说明
基础伤害每次攻击伤害 = 5 + 臂力 × 1.5，再加上技能/武器伤害
武器协同装备武器并学会对应武器技能可额外加伤害。例如 铁剑 + 基本剑法
连招系统连续用同一技能命中敌人，伤害最高叠加 +50%；被招架、闪避或逃跑会重置
战力爆发exert powerup 可在 30 秒内提升 30% 伤害
wear 黑风刀             # 装备稀有武器
learn 基本刀法 × 10     # 学会并提升对应武器技能
perform 基本刀法.断岳斩   # 施展绝招
