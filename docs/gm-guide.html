管理服务器、维护秩序、调试内容的命令参考

🔑 第一章：获得管理员权限

GM 命令只对 isAdmin = true 的玩家生效。目前需要在服务器数据文件中手动设置：

# 1. 让目标玩家先登录一次，使其数据写入 server/data/players.json
# 2. 停止服务器（避免存档覆盖）
# 3. 编辑 server/data/players.json，在该玩家对象中加入：
"isAdmin": true
# 4. 重新启动服务器
安全提示：不要给普通玩家随意开启管理员权限。目前认证系统主要依赖账号密码，GM 账号请使用强密码。

🎛 第二章：GM 命令一览

所有 GM 命令均以 gm 开头：

命令参数作用
gm list无列出当前在线玩家及其所在房间
gm inspect <玩家名>玩家名查看指定玩家的完整状态（气血、内力、经验、潜能、善恶值、装备等）
gm kick <玩家名>玩家名将指定玩家移出游戏
gm goto <roomId>房间 ID传送到任意房间，例如 gm goto wilderness/cave
gm spawn <npcId>NPC ID在当前房间召唤一个该 NPC 的克隆体，例如 gm spawn bandit
gm set <玩家名> <字段> <值>玩家名、字段、值修改玩家属性或状态

📝 第三章：gm set 可修改字段

字段说明示例
hp气血gm set 张无忌 hp 100
mp内力gm set 张无忌 mp 200
exp经验gm set 张无忌 exp 1000
pot潜能gm set 张无忌 pot 500
shen善恶值gm set 张无忌 shen 0
room当前房间 IDgm set 张无忌 room town/square
str / int / con / dex / per / kar六维属性gm set 张无忌 str 20
提示：修改属性后会自动重新计算气血/内力上限；修改 room 时会校验房间是否存在。

🐉 第四章：常用 GM 工作流

4.1 快速巡查在线玩家

gm list
# 输出示例：张无忌 (zhangwuji) — town/square

4.2 调查可疑玩家

gm inspect 张无忌

4.3 踢出捣乱玩家

gm kick 捣乱者

4.4 举办活动 / 测试战斗

gm goto wilderness/cave
gm spawn boss-heifeng
# 召唤一个黑风寨主克隆体用于测试或活动

4.5 修正玩家异常状态

gm set 张无忌 hp 1
gm set 张无忌 room town/square

⚠ 第五章：注意事项

  • GM 命令执行结果只返回给执行者，不会被其他玩家看到。
  • gm spawn 召唤的是 NPC 克隆体，ID 带有时间戳，不会覆盖原 NPC。
  • 修改玩家数据会触发自动存档；如需批量修改，建议直接编辑 server/data/players.json 并重启服务器。
  • 目前不支持运行时动态授予/撤销管理员权限，必须通过数据文件修改。
