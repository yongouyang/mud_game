# 炎黄群侠传 — 商业化支付方案评估

> **评估日期**: 2026-06-22
> **评估目标**: 为 mud_game 设计元宝充值 + 月卡订阅 + 丹药/活动礼包购买的完整支付方案

> - 商品: **元宝（通用充值货币）+ 月卡 + 永久丹药 + 限时活动礼包**

## 1. 商户号注册条件与流程

### 微信支付商户号

**注册条件**（个人/个体工商户均可）:
- 中国大陆身份证 + 手机号 + 同名银行卡
- 经营场景说明 + 网站截图（ICP备案非强制，但建议有）
- 个体工商户营业执照（个人可申请，费率稍高）

**注册流程**: 微信支付商户平台 pay.weixin.qq.com 注册 → 提交身份信息 → 1-3工作日审核 → 获取 mchId + API v3 Key → 下载商户证书 → 对接 API

**费率**: 0.6% | 结算周期 T+1

**关键结论**:
- Native 支付（扫码付）: ✅ 无需域名备案，**推荐首选**
- JSAPI 支付（公众号内）: ❌ 需要认证公众号 + 备案域名
- H5 支付（浏览器内）: ❌ 需要企业资质 + 备案域名

### 支付宝商户号

**注册流程**: 支付宝开放平台 open.alipay.com → 创建应用 → 签约"电脑网站支付" → 获取 APPID + 密钥

**费率**: 0.6% | PC网站支付

### ⚠️ 最大障碍：域名备案与 HTTPS 回调

| 产品 | 是否需要备案域名 | 可行性 |
|------|-----------------|--------|
| 微信 Native 支付（扫码付） | ❌ 不需要 | ✅ **推荐首选** |
| 支付宝当面付（扫码付） | ❌ 不需要 | ✅ **推荐首选** |
| 微信 JSAPI / H5 支付 | ✅ 需要 | 🟡 需先备案 |
| 支付宝电脑网站支付 | ✅ 需要 | 🟡 需先备案 |

> 支付回调必须走 HTTPS。解决方案: **AWS ECS + ALB**（ALB 自带 ACM 免费 HTTPS 证书），域名买海外的无需备案。

## 2. 支付产品选型: 扫码付（Native 支付）

```
玩家浏览器                 mud_game Server              微信/支付宝
  │                           │                            │
  │ 输入 "充值 30"             │                            │
  ├──────────────────────────►│                            │
  │                           ├─ POST /v3/pay/native ────►│
  │                           │◄── {code_url: "weixin://..."}
  │ {"qrCodeUrl", orderId}  │                            │
  │◄──────────────────────────┤                            │
  │                           │                            │
  │ (玩家手机扫码支付...)       │                            │
  │                           │   POST /api/payment/notify│
  │                           │◄───────────────────────────┤ (异步回调)
  │                           ├─ 验证签名 → 发放元宝         │
  │◄── socket.emit("充值成功")──┤                            │
```

**为什么选扫码付**: MUD 玩家电脑上打游戏，手机扫码支付这种体验在这个圈子非常自然。且免去了域名备案的等待周期。

## 4. 虚拟货币与商品体系设计

### 4.1 货币层级

```
元宝 (YuanBao)  ←── 人民币充值 (唯一来源)
  ├── 兑换商店: 永久丹药 / 限时称号 / 转世道具 / 活动礼包
  ├── 购买月卡: 30天特权
  └── 兑换银子: 100元宝 = 100两银子

银子 (Silver)  ←── 游戏内赚取 / 元宝兑换
  └── 游戏内商店 (武器/防具/药品)

潜能 (Pot)    ←── 纯游戏内获取 (不可购买)
```

### 4.2 防止 Pay-to-Win 原则

| 原则 | 说明 |
|------|------|
| 元宝不直接买属性 | 买便利/加速/外观，不买战斗力 |
| 丹药有限且昂贵 | 每天限购 1 颗（月卡 3 颗），防止氪金碾压 |
| 核心战力靠时间 | 武功等级、战斗经验只能游戏中获得 |
| 月卡是性价比之王 | 鼓励订阅而非单次消费 |

## 5. 定价策略建议

### 5.1 行业参考

| 游戏 | 月卡 | 元宝比例 | 模式 |
|------|------|---------|------|
| 炎黄MUD（非官方服） | 30元/月 | — | 丹药/转世道具 |
| 文字修仙类手游 | 25-68元/月 | 1:10 | 月卡+基金 |
| 主流手游 | 30-98元/月 | 1:10 | 混合制 |

### 5.2 推荐定价: 1元 = 10元宝

**理由**: 方便心算；属性丹 500元宝=50元，永久效果但克制；月卡30元性价比最高。

### 5.3 ARPU 估算（假设 100 DAU 小众 MUD）

| 玩家分层 | 比例 | 月消费 | 月收入 |
|---------|------|--------|--------|
| 免费玩家 | 60% | ¥0 | ¥0 |
| 月卡党 | 25% | ¥30 | ¥750 |
| 小R | 10% | ¥50-100 | ¥500-1000 |
| 中R | 4% | ¥200-500 | ¥800-2000 |
| 大R | 1% | ¥1000+ | ¥1000+ |
| **总计** | | | **¥3000-5000/月** |

> MUD 小众但用户粘性高，付费转化率通常 15-25% vs 普通手游的 5%。

## 7. 后端 API 设计

### 对外部（支付回调，必须 HTTPS）

```
POST /api/payment/notify/wechat  ← 微信支付回调
POST /api/payment/notify/alipay  ← 支付宝回调
```

### 对游戏客户端

```
POST /api/payment/order          → 创建订单 → 返回二维码URL
  Body: { productId: "yuanbao_6" }
  Response: { orderId, qrCodeUrl, amount, expireAt }

GET  /api/payment/order/:orderId → 查询订单状态 (前端轮询用)
GET  /api/payment/products       → 获取充值产品列表

POST /api/payment/spend           → 元宝消费
POST /api/payment/monthly-claim   → 月卡每日签到
```

### 游戏内命令

```
> 充值   |  > 充值 30  |  > 元宝  |  > 商城  |  > 购买 力量丹  |  > 月卡  |  > 签到
```

## 8. 安全与防刷

| 威胁 | 防护 |
|------|------|
| 伪造回调 | 验证微信/支付宝 RSA-SHA256 签名 |
| 重复回调 | channelOrderId 幂等：同一笔只处理一次 |
| 金额篡改 | 用 out_trade_no 查本地订单，对比金额 |
| 二维码劫持 | 前端只展示服务器返回的 qrCodeUrl |
| 元宝负数 | 消费时检查 yuanbao >= price |
| 并发消费 | Node.js 单线程安全，async 竞态加锁 |
| 刷元宝 | 订单状态机 pending→paid 不可逆 |
| CSRF | 充值 API 验证 session |

## 11. 合规风险提醒

| 风险 | 严重度 | 缓解措施 |
|------|--------|---------|
| ICP 备案缺失 | 🔴 高 | 海外服务器 (AWS 新加坡) 规避 |
| 游戏版号 | 🔴 高 | 小圈子内测；挂"技术实验"性质 |
| 虚拟货币合规 | 🟡 中 | 元宝不支持提现/退款，单向充值 |
| 税务 | 🟡 中 | 月入超 5000 建议注册个体工商户 |
| 支付风控 | 🟡 中 | 小规模运营（<10万/月流水）风险低 |

**合规清单**:
1. ✅ 仅单向充值，元宝不可提现
2. ✅ 项目标注"技术学习交流，非商业运营"
3. ✅ 海外服务器规避 ICP 备案
4. ✅ 单笔上限 ¥648，单日上限 ¥2000
5. ✅ 不公开宣传，仅目标圈子传播
6. ✅ 充值页面加"请理性消费，未成年人需监护人同意"
