📋 第一章：初入江湖

1注册和创建角色 
register 张无忌 123456    # 注册一个账号
login 张无忌 123456       # 登录已有账号

或者直接用演示账号：login demo some-secret（预设属性，10000潜能）

2分配属性

创建角色后，输入角色名（必须是中文），然后分配 10点自由属性：

set str 20    # 臂力（影响攻击力 + HP上限）
set con 20    # 根骨（影响HP上限 + 防御）
done          # 确认完成创建
推荐新手上路加点： str=15, con=15（高攻击+高血量，打怪不死）
3熟悉基本命令 
look          # 查看四周（会显示地上物品）
hp / score    # 查看状态（HP/MP/经验/潜能/Lv等级）
help          # 显示所有命令
skills        # 查看已学武功
i             # 查看背包物品
who           # 查看在线玩家

📈 第二章：学习武功

武功分两类：基本武功随处可学，独门武功需先拜师。

🔹 基本武功（消耗 1 潜能/次）

武功名ID用途
招架parry基本招架之术，格挡敌人攻击。
基本剑法sword基本剑术，可用剑类武器施展。
基本刀法blade基本刀术，可用刀类武器施展。
基本杖法staff基本棍杖之术。
基本暗器throwing基本暗器手法。
基本鞭法whip基本鞭法之术。
基本拳脚cuff基本拳脚功夫，入门必学。
基本轻功dodge基本轻功，用于闪避敌人攻击。
基本内功force基本内功心法，影响内力上限和护体效果。
醉拳zui-quan形醉意不醉，步醉心不醉的奇门拳法。
learn 基本拳脚    # 学一级拳脚（也可以用英文ID learn cuff）
learn 基本拳脚    # 重复输入升级！每次升1级，最高100级
升级技巧：用 ↑↓ 箭头键调出历史命令，快速重复 learn！

🔹 独门武功 — 先拜师！

高级武功需要：① 已加入对应门派 ② 在师父房间 ③ 基础武功达标

🔹 练功顺序推荐

# 第1步：在广场练基本功（消耗潜能，打山贼获得）
learn 基本拳脚    × 10次 → Lv.10
learn 基本轻功    × 5次  → 提高闪避
learn 基本内功    × 5次  → 提高内力吸收

# 第2步：去少林拜师（路径：n n n e s w）
join 少林派

# 第3步：学独门武功（在师父面前）
learn 罗汉拳      → 消耗2潜能，需要拳脚≥Lv.5
learn 太极拳      → 消耗2潜能，需要拳脚≥Lv.10

⚔ 第三章：打怪升级

山林中有主动攻击的野怪，杀死获得经验+潜能+银两。

野怪ID位置路径难度
山贼bandit山林·入口nnn易
野狼wolf山林·深处nnnn中
黑熊bear山林·洞穴nnnnw难
# 打山贼（新手首选）
n n n              # 走到山林入口（路径 n n n）
kill 山贼           # 发起战斗（也可以用英文ID kill bandit）
# 自动回合制战斗，等几轮后山贼倒下
# 获得经验和潜能！

# 逃跑（打不过就溜）
flee / tao
⚠ 注意：战死后会扣 10% 经验！血少的时候先用 exert heal 疗伤（消耗30内力，恢复20%HP）。如果内力不足，远离战斗等自动恢复（每3秒恢复4%MP）。

🛡 第五章：装备与恢复

🎯 第六章：任务系统

quest 王掌柜         # 向王掌柜接任务（在客栈）
quest 王掌柜         # 再次找同一NPC完成任务，获得经验+潜能奖励
提示：接任务和交任务都是 quest NPC名。任务类型为送信，多刷任务可获得大量经验。

⚡ 第七章：进阶技巧

绝招（perform）

# 武功升到 Lv.10 后可施展绝招
learn 基本拳脚    × 9次（升到Lv.10）
perform cuff.quan # 施展基本拳脚的绝招（消耗20内力，伤害3倍）

内功运用（exert / yun）

exert heal        # 疗伤：消耗30内力，恢复20%HP
yun heal          # yun 是 exert 的别名
exert powerup     # 战力爆发：消耗50内力，短时间内攻防提升

独门绝招

各门派独门武功有更强的绝招（在师父房间学得）。学到Lv.10后 perform 武功名.招式名 施放。
